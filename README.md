## ‚ú® Key Features

## üöÄ Getting Started

### Prerequisites
- A modern web browser.
- A Google Gemini API Key (for the AI Consultant).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/eduglide-global.git
   ```
2. Navigate to the project root:
   ```bash
   cd eduglide-global
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Configuration
The app requires an environment variable for the Gemini API:
- `process.env.API_KEY`: Your Google Gemini API Key.

### Running the App
```bash
npm run dev
```
